What It Does

Mythling EPG is a streamlined, mobile-first program guide for MythTV. From your browser, Mythling EPG gives you access to all your upcoming TV listings. From there you can search for programs, schedule recordings, and even initiate transcode jobs.

How To Use It

You need MythTV 0.27 or later. Click the download button above and unpack the distribution zip file into the root directory of your backend's built-in web server. That's all there is to it! Now point your browser to:
http://<backend-ip>:6544/mythling-epg/guide.html (or guide-omb.html for older browsers).

If you have the Mythling mobile app for Android or Fire TV, then you already have Mythling EPG embedded in the app. So you don't need to install anything to access the guide from within Mythling (unless you want the latest development build).
