Mythling

Mobile Client for MythTV

Getting Started

If you have MythTV and an Android device, you can use Mythling to browse and stream your recordings and media collection. The quickest way to start using Mythling is to install it from the Play Store and follow the setup guide in the online help.

On the Play Store

https://play.google.com/apps/testing/com.oakesville.mythling

On GitHub
